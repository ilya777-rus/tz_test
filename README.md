## Запуск тестов

В репозитории предусмотрены три скрипта для запуска автоматических тестов:

### Команды

Запуск 1 скрипта:

```bash
pytest -s -v --tb=line  test_script1.py
```
Запуск 2 скрипта:

```bash
pytest -s -v --tb=line  test_script2.py
```
Запуск 3 скрипта:

```bash
pytest -s -v --tb=line  test_script3.py
```
